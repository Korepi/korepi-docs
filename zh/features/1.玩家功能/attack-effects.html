                                                                                # 倍功模式:

                                                                                • 一拳模式：可以用1次攻击杀死目标.

                                                                                注意：不建议使用.

                                                                                • 修改目标受到的攻击次数。

                                                                                注意：不建议超过“5”。

                                                                                # 多个目标：

                                                                                • 击中目标命后在其半径内的所有目标受到伤害。

                                                                                注意：不建议更改默认设置。

                                                                                # 多重动画：

                                                                                • 修改角色攻击动画。

                                                                                注意：不建议使用。

                                                                                # 攻击速度：

                                                                                • 修改玩家攻击的速度。

                                                                                注意：不建议使用高于“1.5”的值。

                                                                                # 自定义元素：

                                                                                • 修改所输出的元素。

                                                                                # 注入元素：

                                                                                • 修改角色普通攻击元素。

                                                                                • 自动攻击弱点，只适用弓箭角色。
